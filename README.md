## Backend

### Tech stack
- Node.JS
- Express
- MongoDB

### Startup
- npm install
- node(mon) app.js (server listens for requests on port 3000)


# Frontend

### Tech stack
- Ionic 6 + Angular
- Typescript, HTML, SCSS

### Startup
- npm install
- npm start (browser opens up on URL localhost:4200)
